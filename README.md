### 安装

##### 创建项目

```bash
composer create-project mitirrli/laravel-api -vvv
```

##### 安装需要的包
```bash
npm i
```

##### 使用docker

```bash
docker-compose up --build -d
```

### 常用命令

##### 快速启动项目

```bash
docker exec -it laravel-api composer serve
```

##### 更新项目

```bash
composer update-project
```

##### 单元测试

```bash
composer test
```

##### 修复代码格式错误

```bash
composer cs-fix
```
